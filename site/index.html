MERT Genre Classifier

Objetivo

Classificar gênero musical sem fine-tuning pesado.

O MERT fica congelado como extrator de representações. Os embeddings são cacheados em disco e alimentam uma regressão logística baseline e uma MLP pequena, tornando os experimentos rápidos depois da primeira inferência.

Pipeline

Da faixa de áudio ao relatório de avaliação.

01

Dataset

Carrega splits públicos, valida colunas e preserva o mapa de labels.

02

Diagnóstico

Inspeciona uma amostra de áudio e confirma a decodificação antes do MERT.

03

Embeddings

Reamostra para 24 kHz, roda MERT congelado e salva vetores em cache.

04

Classificador

Treina baseline linear e MLP sobre os vetores já extraídos.

05

Avaliação

Gera macro-F1, accuracy, matriz de confusão e relatório por gênero.

Execução

Smoke test local, extração completa em CUDA quando fizer sentido.

O gargalo esperado é a extração dos embeddings. Por isso o projeto tem uma configuração pequena para validar tudo localmente e outra pronta para rodar o dataset completo numa máquina com GPU.

CPU smoke Colab/CUDA Cache retomável Artefatos portáveis AudioDecoder 
python -m mert_genre_classifier -c configs/local_smoke.yaml prepare-data
python -m mert_genre_classifier -c configs/local_smoke.yaml extract-embeddings --resume
python -m mert_genre_classifier -c configs/local_smoke.yaml train
python -m mert_genre_classifier -c configs/local_smoke.yaml evaluate
python -m pip install -e ".[dev]"
python -m mert_genre_classifier -c configs/full_cuda.yaml prepare-data
python -m mert_genre_classifier -c configs/full_cuda.yaml inspect-audio --split train --index 0
python -m mert_genre_classifier -c configs/full_cuda.yaml extract-embeddings --split train --resume
python -m mert_genre_classifier -c configs/full_cuda.yaml extract-embeddings --split test --resume
python -m mert_genre_classifier -c configs/full_cuda.yaml predict caminho/para/musica.wav --model mlp --top-k 5
Resultados esperados

O projeto já nasce preparado para mostrar qualidade e diagnóstico.

Macro-F1 como norte

A métrica principal evita que gêneros frequentes escondam classes mais difíceis ou desbalanceadas.

Matriz de confusão

Os relatórios em CSV ajudam a ver quais gêneros se confundem e onde melhorar o dataset ou o classificador.

Predição top-k

A CLI retorna probabilidades ordenadas, útil para demonstrações e análise de faixas ambivalentes.
